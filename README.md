## Quickstart

### Step 1 : Clone this repository

```bash
git clone git@github.com:xebia-france/xskillz-v2.git
cd xskillz-v2
```

### Step 2 : Retrieve and build

```bash
docker-compose up --build
```

Have a coffee, it takes some time.
